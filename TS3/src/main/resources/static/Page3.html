Long Island Iced Tea


Aunque se llame "Iced Tea" este coctel no tiene ningún ingrediente con té. Tiene Fama de ser un coctel sumamente fuerte, pero suave.


Se prepara con los siguientes ingredientes:


  • Vodka 3/4oz
  • Ron Blanco 3/4oz
  • Tequila Blanco 3/4oz
  • Ginebra 3/4oz
  • Triple Sec 3/4oz
  • Jarabe de Azucar 3/4oz
  • Jugo de Limón 3/4oz
  • Coca Cola
  • Gajo fino de Limón

Procedimiento:


  1. Se mezcla el vodka, ron, tequila, gin, triple sec, azucar y limón en un vaso alto con hielo
  2. Se termina de llenar el resto del vaso con coca cola y se revuelve ligeramente
  3. Se adorna con el gajo fino de limón y se sirve con una pajilla.
